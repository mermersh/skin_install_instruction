注意事項

  • 4Dスキンが使用可能な端末は、

    Windows

    iPhone

    iPad

    Android


    に限られます。

    ※その他端末・OSで使用する方法があるかもしれませんが、めるんは対応できません。
    ※めるんはAndroid端末を持っていないため、Androidでの動作確認はできかねます。

  • 4Dスキンは、マーケットプレイスにて無料で配布されている『1周年記念スキンパック』を置き換える形で実装します。導入後は『1周年記念スキンパック』は使用できなくなりますのでご了承ください。（元に戻すことは可能です）
  • 一部のサーバーでは、サーバー側の設定により使用できない場合があります。また、特定の環境では使用できない場合があります。
  • 4Dスキンの導入は公式でサポートされていません。導入に伴って不利益が生じた場合やその他のトラブルに対して、めるんは一切の責任を負いかねます。
  • 基本的には初回に一度導入するだけですが、アップデートなどで再導入が必要になる場合があります。その場合も、初回導入時と同じ手順で進めてください。
  • 今後のマインクラフト統合版のアップデート・仕様変更等により、4Dスキンが使用できなくなる可能性があります。Microsoftさんの気分次第なので、お許しください。
  • 導入方法についての質問やサポートは、めるんまでご連絡ください。可能な限りお答えします。

導入方法

※導入したい端末を選んでください

Windows

方法A：Skin Injectorを使用する（おすすめ）
  1. このページの「リンク」から Skin Injectorダウンロードサイト へアクセスし、最新版の Skin Injector をダウンロードする
  2. ダウンロードした Skin Injector を分かりやすい場所（デスクトップ等）に置く
  3. Skin Injectorの 使い方ページ を参考に導入を進める

    ※Step1でダウンロードしておくスキンパックは『1周年記念スキンパック』です。


    ※Step5では、お渡ししたファイルの中の スキン(Windows専用) をフォルダごとドラッグ＆ドロップし、下のスキンパック一覧の中から『1st Birthday Skin Pack』を選択したうえで、Injectボタンをクリックしてください。

  4. Minecraft統合版を起動し、スキン選択画面で1周年記念スキンパックが正常に置き換わっていればOK!
    そのまま選んで使えます (*>ω<)b

うまくいかない場合

うまくいかない場合は以下の手順を試してください。

  1. 方法Bの『Windows版でskin_packsフォルダの場所にアクセスする方法』に従って、skin_packsフォルダの場所にアクセスする
  2. premium_cache/skin_packs フォルダの中身をすべて消す
  3. 統合版を再起動し、スキン選択画面に1周年記念スキンパックがあることを 必ず 確認する
  4. 統合版を閉じて、もう一度導入の手順を行う
方法B：手動で導入する
  1. このページの「リンク」から iOS用解説動画 へアクセスする
  2. 手順に沿って進める
    ファイルをコピーする作業の前に、統合版を閉じてください。

    ※動画ではiOSで操作していますが、同様の手順でWindowsでも導入が可能です。

    ※ダウンロードしておくスキンパックは『1周年記念スキンパック』です。

    ※コピーするスキンパックは、お渡ししたファイルの中の スキン(手動置き換え用・iOS,Android,Windows対応) です。

    Windows版でskin_packsフォルダの場所にアクセスする方法

    1. デスクトップでWinキーとRキーを同時に押す
    2. 『ファイル名を指定して実行』が開いたら、 %appdata% と入力してEnterキーを押す
    3. 開いたフォルダの中にある Minecraft Bedrock フォルダを開く
    4. premium_cache フォルダを開く
    5. ここに skin_packs フォルダがあるはず！

  3. Minecraft統合版を再起動し、スキン選択画面で1周年記念スキンパックが正常に置き換わっていればOK!
    そのまま選んで使えます (*>ω<)b

統合版（Switch / PlayStation / Xbox）

大事：多くの場合、端末単体ではPNG取り込みができません
  • コンソール版は「ローカル画像からのスキン読み込み」が制限されることがあります
  • 基本は PC/スマホ（統合版）で取り込み → Microsoftアカウントで同期 が安定です
おすすめ手順（同期ルート）
  1. PC または スマホの統合版で skin.png を取り込んで適用
  2. 同じ Microsoftアカウント で Switch/PS/Xbox 側もログイン
  3. 更衣室を開いてスキンが出るか確認（反映に時間がかかる場合あり）
それでも出ない場合
  • 一度ゲームを再起動
  • サインアウト → 再サインイン
  • 更衣室のキャッシュが更新されるまで待つ（数分〜）
メモ： 端末・バージョンによって挙動が違うので、状況が分かれば案内文をあなた用に最適化できます。

Java版（PC）

公式プロフィールで変更する（基本）
  1. skin.png を保存
  2. Minecraft Launcher の Skins から追加、または公式プロフィールページでアップロード
  3. モデル（スリム/クラシック）を選んで保存
注意
  • Java版はオンラインプロフィール反映に少し時間がかかることがあります
  • 非公式ランチャー/オフライン環境は別手順になる場合があります
ポイント： 受け取ったスキンが「統合版向け仕様」だと、Javaで見え方が変わることがあります。
